Section 4.1 forms.checkboxes Checkboxes

If an <input> element has the type='checkbox' attribute set, the form field is displayed as a checkbox.

It's recommended that you don't attempt to style these elements. Firefox's implementation doesn't respect box-sizing, padding, or width.

Example
Weekdays 
<fieldset class="c-form__checkboxes">
  <legend><span class="c-form__label">Weekdays</span></legend>
    <ul class="c-list--naked">
      <li class="c-form__item -checkbox">
        <input class="c-form__checkbox" id="monday" type="checkbox" name="monday" value="monday" checked="">
        <label class="c-form__label -option" for="monday">Monday</label>
      </li>
      <li class="c-form__item -checkbox">
        <input class="c-form__checkbox" id="tuesday" type="checkbox" name="tuesday" value="tuesday">
        <label class="c-form__label -option" for="tuesday">Tuesday</label>
      </li>
      <li class="c-form__item -checkbox">
        <input class="c-form__checkbox" id="wednesday" type="checkbox" name="wednesday" value="wednesday">
        <label class="c-form__label -option" for="wednesday">Wednesday</label>
      </li>
      <li class="c-form__item -checkbox">
        <input class="c-form__checkbox" id="saturday" type="checkbox" disabled="">
        <label class="c-form__label -option" for="saturday">Saturday</label>
      </li>
    </ul>
</fieldset>

Section 4.6 forms.labels Label

The <label> element represents a caption of a form field. The label can be associated with a specific form control by using the for attribute or by putting the form control inside the label element itself.

Example 
<div class="c-form__item">
  <label class="c-form__label " for="labelInputText">A label for a text input <span class="c-form__required" aria-label="Required" title="Required">*</span></label>
  <input type="text" id="labelInputText" placeholder="Enter some text">
</div>

<div class="c-form__item">
  <label class="c-form__label -option"><input type="checkbox" /> A label wrapped around a checkbox</label>
</div>

Section 4.8 forms.select Select list

The <select> element represents a form field for selecting amongst a set of options.

Known limitation: by default, Chrome and Safari on OS X allow very limited styling of <select>, unless a border property is set.

Examples
