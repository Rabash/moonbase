Section 7 grid Grid

The grid is based on 12 columns for breakpoints large and up. Smaller breakpoints uses a 2 column layout. The @-sign is used to determine at what breakpoint this grid is used. An element with the classes o-1-2@xsmall-large and o-8-12@large will be 1/2 of the available screen size at breakpoints small to large, and 8/12 of the available size at breakpoints large and up.

Example

8/12 and 4/12 width at breakpoint large:

.o-8-12@large .u-left@large
.o-4-12@large .u-left@large .u-last@large

4/12 width at breakpoint large:

(Because all items in row are equal with, the u-last@large class can be omitted.)

.o-4-12@large .u-left@large
.o-4-12@large .u-left@large
.o-4-12@large .u-left@large

6/12 width at breakpoint large, 3/12 width at breakpoint xlarge

.o-6-12@large .o-3-12@xlarge .u-left@large
.o-6-12@large .o-3-12@xlarge .u-left@large
.o-6-12@large .o-3-12@xlarge .u-left@large
.o-6-12@large .o-3-12@xlarge .u-left@large

1/2 width at breakpoint medium-large, 8/12 and 4/12 at breakpoint large:

.o-1-2@medium-large .o-8-12@large .u-left@medium
.o-1-2@medium-large .o-4-12@large .u-left@medium

Section 7.1 grid.row Row

Rows are used to create a consistent system of margins between elements.

Example
.o-row
This is a regular row. Margin at bottom.
.o-row.-small
This is a small row. Margin at bottom.
.o-row.-large
This is a large row. Margin at bottom.

Section 7.1.1 grid.row.faux Faux Row

Faux rows are used like rows, but for places that need top padding instead. Useful for first elements in wrappers. Often used together with o-row. Always consider using a row above the element instead of a faux row.

Example
.o-row.o-faux-row
This is a regular row. Padding at top.
.o-row.o-faux-row.-small
This is a small row. Padding at top.
.o-row.o-faux-row.-large
This is a large row. Padding at top. 
<div class="o-wrapper kss-wrapper">
 <div class="o-row o-faux-row kss-row">
  <code>.o-row.o-faux-row</code><br/>This is a regular row. Padding at top.
 </div>
 <div class="o-row o-faux-row -small kss-row">
   <code>.o-row.o-faux-row.-small</code><br/>This is a small row. Padding at top.
 </div>
 <div class="o-row o-faux-row -large kss-row">

Section 7.2 grid.trumps Grid utilities

Helpers for grid. Overrides all other properties with !important. Should be avoided if possible. The @-sign is used to determine at what breakpoint this grid is used.

Parameters:
  • .u-left
    Float element to the left
  • .u-right
    Float element to the right
  • .u-last
    Removes right margin on grid elements
  • .u-center
    Center with margin: 0 auto;
  • .u-center-align
    text-align: center;
  • .u-center-vertical
    Vertical centering
  • .u-clear-none
    clear: none;
  • .u-first
    clear: left;
  • .u-overflow-hidden
    overflow: hidden;
  • .u-relative
    position: relative;
